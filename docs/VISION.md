# Vision — die stehende Stimme

## Prinzip 1: Passive Twins sind Bürger erster Klasse

Ein Twin ist eine **stehende Stimme**, kein Abo auf Aktivität. Wer einmal
trainiert und teilt, wirkt ab dann ohne jedes Zutun: Der Twin wird in jeder
Auswertung gezählt, beantwortet per Inferenz automatisch jede künftige Frage
und hält sein Land auf der Karte am Leuchten. Einmal im Jahr vorbeischauen
ist ein vollwertiger Nutzungsmodus — die Tagesfrage ist ein Angebot, nie
eine Pflicht.

**Folgeprinzip Frische:** Veralterung wird transparent gemacht, nie
sanktioniert. Auswertungen weisen perspektivisch das Alter der zugrunde
liegenden Twins aus („Ø-Alter der Stimmen"); eine sanfte jährliche
Erinnerung („Stimmt dein Zwilling noch?" → 30-Sekunden-Kalibrierung)
ersetzt jeden Aktivitätszwang.

## Prinzip 2: Die Stimme als Interface (Phase 2)

Sprechen ist die einzige Schnittstelle, die jeder Mensch beherrscht —
einschließlich der ~750 Millionen Menschen ohne Lese- und Schreibkenntnisse,
die jedes Formular der Welt ausschließt. Ziel: Der Twin entsteht im Gespräch.

Die Treppe dorthin:

1. **Vorlesen (jederzeit baubar):** Fragen per Sprachausgabe — läuft lokal
   auf jedem Gerät, keine KI nötig.
2. **Spracheingabe (nah):** Sprache→Text auf dem Gerät (Whisper-Klasse,
   ~40 MB im Browser) für gesprochene Antworten auf bestehende Fragen.
3. **Das Gespräch (Vision):** „Erzähl mir, was dich bewegt" — ein Modell
   **auf dem Gerät** übersetzt freie Sprache in Dimensions-Vorschläge, die
   der Mensch bestätigt.

**Rote Linie:** Politische Äußerungen verlassen niemals das Gerät. Ein
Server-LLM ist für diesen Zweck ausgeschlossen — die Gesprächs-Stufe wartet
auf alltagstaugliche On-Device-Modelle, nicht auf einen bequemen API-Key.

*Eine Stimme in der Welt — wörtlich.*
